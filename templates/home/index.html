About this website

This is a fully responsive website making use of svg, responsive images, high res png for retina display, google fonts.
It uses relative units like vmin, em, percentages for scaling acording to the device display size (no media queries).
Pure css with sass for pre-compilation and no framework like bootstrap is used for styling.
Angularjs is used for routing and jquery for event handling.
